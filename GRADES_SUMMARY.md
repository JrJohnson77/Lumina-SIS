## 🎯 How to Access Grades in the App

### Via Gradebook Page
1. Login with admin or teacher credentials
2. Navigate to **Gradebook** from sidebar
3. Select a class (e.g., "SUNF-Grade 3A")
4. Choose **Term 1** and **Academic Year 2024-2025**
5. View complete grade table with all students and subjects
6. Click individual cells to see component breakdown

### Via Student Profile
1. Go to **Students** page
2. Click any student name
3. View their complete academic record
4. See all subject grades and overall performance

### Via Reports Page
1. Navigate to **Reports**
2. Select a student
3. Choose **Term 1, 2024-2025**
4. Generate report card with all grades included
